Linked List (Data Structure)

Create a linked list using Test First or Test Driven Development. Assume all values in the linked list will be unique. Linked List documentation can be found here.

  1. Start with a single linked list that implements the ICollection<int> interface.
  2. Modify the implementation support any data type.
  3. Add a second implementation of a doubly linked list.
